## Adding challenge
Go to [randomizer.js](randomizer.js) and add your challenge to *text_c* array, no tide pods and stupid challenges plz.

```
var text_c = [
  'Buy pizza for your friend',
  'HARRY POTTER MARATHON!!!!!',
  'Imitiate Trump *hands gestures*',
  'YOUR CHALLENGE HERE'
];
```
